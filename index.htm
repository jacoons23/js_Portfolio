    The THEMES Section allows you to change the color Theme (Background Color) of the website. On this website, all "containers" (sections, boxes) are based on the base HTML color. By using transparency, changing the HTML color will cause all the colors on the website to change to a color-coordinated color Theme.

    It demonstrates CSS, HTML, Flex containers, flex buttons, dynamic hyperlinks, single-element array, Array.Mapping, dynamic HTML, innerHTML Replacement and event handling.

  • It uses element styles to hide/unhide sections or change their appearance.
